Site Plan

Site Name

Daily Quiz Maker, this name is simple and to the point describing the website's purpose.

Site Purpose

A site dedicated to allowing easy creation of quiz like forums to test yourself every day!

Scenarios

Here are the following questions that users may ask when visiting the site for the first time.

  • Where can I make a quiz?
  • How can I view my previous attempts?

Color Schema

I will be using the navy accent plus a mix of dark and light gray colors.

Typography

This site will use Montserrat, Sans Serif, and Arial.
